# Smart Employee Leave & Attendance Management System

A Java Full Stack web application developed using **Spring Boot, React.js, and MySQL**.

## Tech Stack

* Java 17
* Spring Boot
* REST API
* MySQL
* React.js

## Features

* Employee management
* Leave request system
* Attendance tracking
* Admin dashboard
* RESTful API integration

## Project Structure

* `backend/` - Spring Boot API
* `frontend/` - React UI
* `database/` - SQL scripts
